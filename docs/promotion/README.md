# Promotion

Marketing + distribution reference for `hl-plugins`. Text-only for now —
no banner images, no demo GIFs.

## Files

| File | Purpose |
|---|---|
| [`copy.md`](./copy.md) | Marketing copy — Discord, X, LinkedIn, GitHub Release, README hero, reply templates |
| [`showcase.md`](./showcase.md) | Prompt gallery — image-gen led, other modalities brief |

## Goals

1. **Discoverability** — npm + GitHub search + social previews land right
2. **One-command install visible** — every surface shows the install command
3. **Image generation as the killer feature** — concrete prompt + result
4. **No telemetry / MIT / open source** — reassurance for dev community
5. **CTA invites replies** — "what other plugins should i wrap next?"

## Out of scope

- Custom logo / wordmark design (text-only for now)
- Banner images, animated demos, social preview artwork
- GitHub Pages site / landing page
- Video walkthrough / YouTube
- Paid promotion / sponsored content
- Translations (English only)
